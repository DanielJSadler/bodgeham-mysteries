# Bodgeham Mysteries

TanStack Start + Convex + Tailwind CSS starter.

## Setup

1. Install dependencies: `npm install`
1. Set up Convex: `npm run convex:dev`
1. Start the app: `npm run dev`

TanStack Start will generate `src/routeTree.gen.ts` on the first dev run.

## Stack

- TanStack Start
- Convex
- Tailwind CSS v4
